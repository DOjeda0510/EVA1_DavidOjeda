# Estación Arcade - Evaluación 1
**Estudiante:** David Ojeda  
**Proyecto:** Sitio Web para Empresa de Videojuegos

## 📝 Descripción
Este proyecto es una plataforma web diseñada para "Estación Arcade". Contiene información detallada sobre videojuegos populares, trailers integrados de YouTube y un sistema de contacto para los usuarios.

## 🚀 Características
* **Sección de Juegos:** Información detallada de títulos como GTA V, Sea of Thieves, etc.
* **Multimedia:** Trailers oficiales funcionando mediante un servidor local.
* **Diseño:** Estructura organizada con HTML5 y estilos personalizados en CSS3.
* **Contacto:** Formulario de consultas para clientes.

## 🛠️ Tecnologías utilizadas
* HTML5 / CSS3
* Visual Studio 2026
* ASP.NET Core (como servidor de archivos estáticos)

## ⚙️ Instrucciones de ejecución
Para visualizar correctamente el sitio y los videos de YouTube:
1. Abrir la solución en **Visual Studio**.
2. Presionar el botón **Play (IIS Express)**.
3. El sitio se abrirá automáticamente en `localhost`.
